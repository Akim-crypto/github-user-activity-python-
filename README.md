Описание
CLI-приложение, которое получает и отображает последние события активности пользователя GitHub прямо в терминале. Всё, что требуется — ввести ник пользователя, после чего программа покажет основные действия пользователя на GitHub: коммиты, создания issue, добавление звёзд и прочее.

Установка
Для запуска необходим Python 3.x.

Клонируйте репозиторий или просто скачайте файл github_activity.py.

Использование
bash
python github_activity.py
Введите имя пользователя GitHub по запросу в терминале. Программа выведет последние события активности.
